## 🚀 Quick Start

### 📋 Prerequisites
- Python 3.7+
- pip package manager

### ⚡ Installation

1. **📥 Clone the repository**
   ```bash
   git clone https://github.com/Ajaykumar868878/portfolio.git
   cd portfolio
   ```

2. **🔧 Create virtual environment**
   ```bash
   python -m venv venv
   
   # Windows
   venv\Scripts\activate
   
   # macOS/Linux
   source venv/bin/activate
   ```

3. **📦 Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **🚀 Run the application**
   ```bash
   python app.py
   ```

5. **🌐 Open your browser**
   Navigate to `http://localhost:5000`

## 🔧 Customization

### 📝 Adding Projects

1. **Use Admin Panel**: Visit `/admin` to manage projects
2. **Add New Project**: Fill in project details, upload images
3. **Configure Tech Stack**: Add technologies used
4. **Set Links**: Add GitHub and demo URLs

### 🎨 Customizing Colors

Modify CSS variables in `static/css/style.css`:
```css
:root {
  --primary-color: #your-color;
  --secondary-color: #your-secondary-color;
  --accent-color: #your-accent-color;
}
```

### 📝 Updating Content

Edit the following files:
- **Hero Section**: `templates/index.html`
- **Contact Info**: Update email and social links
- **About Section**: Professional summary
- **Skills**: Add/remove skills as needed

## 🌐 Deployment Guide

### 🚀 Render (Recommended)

1. **📝 Create Render Account** at [render.com](https://render.com)
2. **🔗 Connect GitHub** repository
3. **⚙️ Create Web Service**:
   - Runtime: Python 3
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `gunicorn app:app`
   - Environment Variables: `RENDER=true`
4. **🚀 Deploy** - Automatic build and deployment

### 📋 Render Deployment Checklist

- [ ] ✅ Create Render account
- [ ] ✅ Push code to GitHub
- [ ] ✅ Create Web Service on Render
- [ ] ✅ Set environment variables
- [ ] ✅ Configure build and start commands
- [ ] ✅ Test deployment
- [ ] ✅ Set up custom domain (optional)
